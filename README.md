# Sales Analytics Platform

An end-to-end sales analytics project using Excel, SQL, Python, and BI tools
(Power BI & Tableau).

## Project Objectives
- Clean and transform raw sales data
- Store structured data in a relational database
- Perform analytical queries using SQL
- Build predictive models using Python
- Visualize insights using Power BI and Tableau

## Tech Stack
- Excel
- SQL (PostgreSQL / MySQL)
- Python (Pandas, NumPy, Scikit-learn)
- Power BI
- Tableau

See folder organization for data, SQL scripts, Python code, dashboards, and documentation.
